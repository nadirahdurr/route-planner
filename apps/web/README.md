# Route Planner Web UI

Next.js (App Router) frontend for the offline-first mission planner. This scaffold includes:

- Tailwind / shadcn-ready styling baseline
- MapLibre GL canvas wired to a demo basemap
- React Query + Zustand dependencies for upcoming agent integration

## Local development

1. Install dependencies via pnpm: `pnpm install`
2. Start the dev server: `pnpm dev:web`
3. Open http://localhost:3000

Agent API wiring and terrain bundle upload workflow are under construction.
